    Steradian()

    new Steradian()
    Steradian.js, line 43

    Creates an empty instance of Steradian. Use an instance of Steradian to construct new Quantities, Units, and Systems, and perform conversions between them.
    Examples
    // this instance comes pre-loaded with SI and Imperial.
const Sr = require('steradian');
    // the main module is also a function.
const Steradian = require('steradian');
// constructed instances are empty and fully customizable.
const Sr = Steradian();
